🛠️ Recursos
React: Biblioteca JavaScript focada em criar interfaces de usuário de alta performance.
ASP.NET Core WebAPI: Framework robusto para desenvolvimento de APIs.
TypeScript: Oferece um ambiente de desenvolvimento mais seguro através da tipagem estática.
Entity Framework Core: ORM da Microsoft que facilita o acesso ao banco de dados.
🚀 Instalação
🔧 Backend (ASP.NET Core WebAPI)
bash
Copy code
cd path/to/api-directory
dotnet restore
dotnet run
🔧 Frontend (React)
bash
Copy code
cd path/to/react-app-directory
npm install
npm start
💼 Uso
Inicie tanto o backend quanto o frontend.
Acesse http://localhost:3000 no seu navegador.
Utilize a interface para adicionar, remover ou atualizar entradas financeiras.
