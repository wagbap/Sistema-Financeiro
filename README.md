- **Entity Framework Core**: ORM da Microsoft.

## 🚀 Instalação

## Backend (ASP.NET Core WebAPI)

cd path/to/api-directory<br>
dotnet restore<br>
dotnet run

## Frontend (React) 


cd path/to/react-app-directory<br>
npm install<br>
npm start
